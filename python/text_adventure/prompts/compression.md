请将当前的冒险进度压缩为一份“世界状态快照（World State Snapshot）”，以便后续剧情连贯。

请严格保留以下维度：

1. 当前处境 (Situational Context): 玩家当前精确的物理位置、环境氛围（如：深夜、暴雨）以及即时的危机或任务。
2. 角色状态 (Character Sheets): 
   - 物品栏 (Inventory): 玩家身上所有已获得的道具、消耗品及其实际用途。
   - 生理/心理 (Vitals): 玩家的伤势、精神状态或剩余体力。
3. 重要决策与伏笔 (Critical Path): 玩家此前做过的关键选择（如：救了那个刺客）以及尚未触发的潜在威胁。
4. NPC 关系网 (NPC Relations): 已相遇角色的名字、他们对玩家的态度（信任/敌对）以及最后一次互动的核心内容。
5. 世界观设定 (World Lore): 对话中刚揭露的、后续必须遵守的世界规则（如：此处的魔法在满月时无效）。

压缩禁忌：

* 禁止使用模糊词汇（如“玩家获得了一些东西”），必须具体到“玩家获得了生锈的铁剑”。
* 保留最后 3 轮对话的原始语气，以维持剧情的转场连贯性。
